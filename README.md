## What I’m working on

### 🛣️ [Speed Creep AI](https://github.com/anishpentyala/speed-creep-ai)
CV pipeline that predicts road-design-induced speeding using Google Street View + SegFormer semantic segmentation + XGBoost. Trained on 306 road segments in Bellevue, WA with crash correlation analysis. React/Leaflet web map with SHAP-based plain-English explanations.
